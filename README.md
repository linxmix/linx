## how to install

dependencies:
- neo4j: [debian/ubuntu instructions](http://debian.neo4j.org/)

```
git clone https://github.com/wolfbiter/linx -b hardfork2
npm install
```

## how to run

```
npm start
```

## how to develop

```
npm run develop
```

## how to test

```
npm run test-server
```

OR

```
npm run test-client
```
